
TXT_to_EXCELparser run procedure

1. Before running the script please install xlrd & xlwt attached setups

2. Open the "TXT_to_EXCELparser.py" and run it.

3. It will ask for input file name with extension & output file name without extension.

4. Folder in which "TXT_to_EXCELpaser.py" script was copied, the excel file will be created in the same folder.
